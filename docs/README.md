# User Guide

## Features 

### Todo Feature
Add a todo task to your list.

## Usage

### `todo <task name>` - Adds the todo to your list of tasks

Describe action and its outcome.

Example of usage: 

`todo buy new shoes`

Expected outcome:

`[T][x] <task name>`

## Usage
